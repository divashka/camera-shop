# Интернет-магазин фотоаппаратов

Pet-проект посвященный реализации функционала настоящего интернет-магазина.

Что реализовано в проекте?
<ul>
  <li>поиск товаров</li>
  <li>фильтрация</li>
  <li>сортировка</li>
  <li>слайдер</li>
  <li>корзина</li>
  <li>модальные окна</li>
</ul>

Проект написан на React + TypeScript.
